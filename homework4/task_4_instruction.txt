## Instruction
# Preparation
 - Install Python 3.9 (https://www.python.org/downloads/)
 - Install pytest `pip install pytest`
 - Clone the repository <path your repository>
 - Checkout branch <your branch>
 - Open terminal
 - Change folder to testing module in the clonned repository
 - Read docs to the testing module i.e. cat <module name.py>
 Doctests are placed in the docstring to the testing module or function
 https://docs.python.org/3/library/doctest.html
# Running doctest
 - Run doctest: pytest --doctest-modules <module name>
 - Output should repeat results in the docstring
